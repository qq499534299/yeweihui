# 业委会宣传 H5 + 飞书多维表格留资系统

扫码 → 浏览业委会宣传页 → 填写留资表单 → 数据自动存入飞书多维表格

---

## 第一步：创建飞书多维表格

1. 打开飞书电脑端，进入 **飞书 → 多维表格**
2. 点击 **新建多维表格**，命名为 `业委会业主信息收集`
3. 默认会有一个「表格视图」和一张数据表
4. 添加以下字段（**字段名必须完全一致**）：

| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 姓名 | 文本 | 业主姓名 |
| 楼栋 | 文本 | 楼栋号，如 3 |
| 单元 | 文本 | 单元号，如 2 |
| 门牌号 | 文本 | 门牌号，如 1506 |
| 完整房号 | 文本 | 自动拼接，如 "3栋-2单元-1506" |
| 手机号 | 文本 | 11位手机号 |
| 参与意愿 | 文本 | 愿意参与筹备/愿意签名支持/先了解一下 |
| 提交时间 | 文本 | 自动填入提交时间 |

5. 复制表格的链接，链接格式为：
   ```
   https://xxx.feishu.cn/base/XXXXXXXX?table=YYYYYYYY
   ```
   - `XXXXXXXX` 是 **app_token**
   - `YYYYYYYY` 是 **table_id**
   - 记下这两个值，后面要用

---

## 第二步：创建飞书应用

1. 打开 [飞书开发者后台](https://open.feishu.cn/app)
2. 点击 **创建企业自建应用**，名称填 `业委会信息收集`
3. 创建后进入应用详情页
4. 左侧菜单 → **凭证与基础信息**，记录：
   - **App ID**（`cli_xxxxxxxxxxxx`）
   - **App Secret**（点击"显示"获取）
5. 左侧菜单 → **权限管理**，搜索并开通以下权限：
   - `bitable:app`（多维表格）
6. 左侧菜单 → **安全设置**，添加服务器 IP（可暂时跳过）
7. 点击右上角 **发布应用** → **创建版本并发布** → 管理员审批通过

---

## 第三步：授权应用访问多维表格

1. 回到多维表格页面
2. 点击右上角 **分享** → **添加协作者**
3. 搜索你刚创建的应用名称（`业委会信息收集`）
4. 添加为 **可编辑** 权限

---

## 第四步：部署到 Vercel（免费）

### 准备工作

1. 注册 [GitHub](https://github.com) 账号（如没有）
2. 将本项目推送到 GitHub 仓库
3. 注册 [Vercel](https://vercel.com) 账号（用 GitHub 登录）

### 部署步骤

1. 打开 [Vercel Dashboard](https://vercel.com/dashboard)
2. 点击 **New Project** → 导入你的 GitHub 仓库
3. 框架会自动识别，直接点 **Deploy**
4. 部署完成后，进入项目 **Settings → Environment Variables**
5. 添加以下四个环境变量：

| 变量名 | 值 |
|--------|-----|
| `FEISHU_APP_ID` | 你的飞书 App ID |
| `FEISHU_APP_SECRET` | 你的飞书 App Secret |
| `FEISHU_APP_TOKEN` | 多维表格的 app_token |
| `FEISHU_TABLE_ID` | 数据表的 table_id |

6. 回到 **Deployments** 页面，点击 **Redeploy** 重新部署使环境变量生效
7. 部署后你会得到一个域名，如 `https://yeweihui.vercel.app`

## 常见问题

**Q: 提交后飞书表格没有数据？**
- 检查 Vercel 环境变量是否正确配置
- 确认飞书应用已发布并通过审批
- 确认应用已被添加为多维表格的协作者
- 查看 Vercel Functions 日志排查具体错误

**Q: 微信内打不开页面？**
- 确认 Vercel 域名没有被微信屏蔽（一般不会）
- 如果被屏蔽，可以在 Vercel 设置中绑定自定义域名

**Q: 数据安全吗？**
- 数据传输全程 HTTPS 加密
- Vercel Serverless 函数不存储数据，直接转发到飞书
- 数据存在你自己的飞书多维表格中，只有你能访问
